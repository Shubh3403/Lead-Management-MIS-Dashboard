# Lead Management MIS Dashboard

## 📊 Project Overview
This project is an Excel-based MIS Dashboard designed to track and analyze lead management and sales performance.

## 🚀 Key Features
- Lead tracking (Converted, Lost, Pending)
- Salesperson performance analysis
- Region-wise revenue insights
- Monthly trend analysis
- Interactive dashboard using slicers

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## 📈 KPIs
- Total Leads
- Conversion Rate
- Total Revenue
- Average Deal Size
